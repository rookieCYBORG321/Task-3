# Contact Management Dashboard
Java Full Stack Internship - Task 3

## Project Overview
A full-stack Contact Management Dashboard built to perform complete CRUD operations.

## Tech Stack
* **Backend:** Java, Spring Boot, Spring Data JPA
* **Database:** MySQL (Hibernate auto-generation)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (Fetch API)

## How to Run
1. Ensure MySQL is running on your machine.
2. Update the `src/main/resources/application.properties` file with your MySQL root password.
3. Run the Spring Boot application via `DashboardApplication.java`.
4. Open your browser and navigate to `http://localhost:8080/index.html`.
